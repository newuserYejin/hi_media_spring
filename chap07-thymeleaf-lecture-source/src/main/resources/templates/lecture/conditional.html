th: each

th: each 가장 많이 사용하는 태그

이름 나이 성별 주소

th:each 는 두번째 매개변수 stat 를 추가하면 반복 상태를 확인할 수 있다.

        index : 0 부터 시작하는 인덱스 값
        count : 1 부터 시작하는 값
        current : 현재 객체의 정보
        odd :  홀수 번째의 여부
        even :  짝수 번째의 여부
        first : 첫번째 데이터 여부
        last : 마지막 데이터 여부
이름 나이 성별 주소 index count size current even odd first last
